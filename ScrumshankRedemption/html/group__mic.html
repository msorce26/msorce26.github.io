Detailed Description

The MikeMic trap is designed to allow access when the user is within the given volume range.

This module was designed to as a game that only allows access once a certain ultrasonic distance value is reached for a select period of time. The module is only armed once the Brainframe module sends an "Arm" signal to the system through NRF wireless communication. A random distance away from the sensor is assigned when armed, and the range of values above and below this distance is decided by the difficulty modifier sent from the Brainframe. Information is relayed to the user by serial message and LEDs, (according to the design of the created circuit).

The MikeMic trap was originally designed to take the output of a microphone and allow access after a certain volume was met for a period of time. Since difficulties were met, an ultrasonic sensor replaced the microphone. A random distance is initialized every time the trap is armed. The difficulty of the trap dictates the range in which the user should keep their hand, or any other object detected by the ultrasonic sensor. The user must keep their hand within this range for a period of time to disarm the trap and allow access. This range and distance are relayed to the user by use of LEDs and serial communication to the laptop. If the user is in range, the green LED will turn on, incrementing a timer towards the 13 second period. If the hand is closer or farther than the designated range, a red LED will turn on to the right or left of the green LED, respectively. When the ultrasonic sensor picks up an object inside or outside of the range, the timer will stop incrementing until the object is back inside the range. Once access is permitted, the module will send a signal, notifying the brainframe of the access.

Warning
Game is designed for the MSP430F5529.
Ultrasonic sensor accuracy decreases at farther ranges.
